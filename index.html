Question

Explain the difference between inline and block elements with example in HTML.

Answer

Inline elements
Inline elements do not start on a new line and only take up as much width as necessary.
They typically flow within the content of a block-level element, allowing other inline elements to sit beside them.
Examples of inline elements: span, a, img, input, button, label, etc.


Block elements
Block elements always start on a new line and take up the full width available.
They create a visual "block" of content and will typically break the flow of content, causing subsequent elements to appear on a new line.

Examples of block elements :div, p, h1 to h6, ul, ol, li, table, form, etc.
